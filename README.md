### -Deck.java 実行-
当クラスは、トランプの生成とシャッフルが正常に行われているかテストができる。

1. ジョーカー抜きのトランプ一式を生成
2. シャッフル（1回目）
3. プレイヤー人数に応じ、端数になるカードを消却
4. ジョーカーを2枚追加
5. シャッフル（2回目）

シャッフルは、カジノディーラーが実際に行っていると言われるものを再現した。
シャッフル：「 リフルシャッフル（いわゆるシャッフル） → ストリップカット（上から数枚ずつをつかみ、落としていく） → リフルシャッフル 」
